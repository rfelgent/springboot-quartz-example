Introduction
===

Example project for setting up Spring Boot with Quartz in a programmatic way.
Project created during discussion of issue "https://github.com/quartz-scheduler/quartz/issues/117"


Starting the application

```
./gradlew bootRun
```

Have look at the in-memory database (username: sa, password: , jdbc-url: jdbc:h2:mem:testdb)

```
http://localhost:10333/h2-console/
```
